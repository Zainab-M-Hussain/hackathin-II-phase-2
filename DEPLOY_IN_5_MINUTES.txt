🚀 DEPLOY YOUR APP IN 5 MINUTES - FOLLOW THESE EXACT STEPS

═══════════════════════════════════════════════════════════════════════════

TIMER START ⏱️ (5 minutes)

═══════════════════════════════════════════════════════════════════════════

STEP 1: VERCEL SIGNUP (1 minute)
─────────────────────────────────

1. Open in NEW BROWSER TAB:
   👉 https://vercel.com/signup

2. Click "Continue with GitHub"

3. GitHub will ask permission - Click "Authorize"

4. You're in! ✅

═══════════════════════════════════════════════════════════════════════════

STEP 2: IMPORT PROJECT TO VERCEL (1 minute)
────────────────────────────────────────────

1. Click "Add New" → "Project"

2. Click "Import Git Repository"

3. Search for your repository:
   → hackathin-II-phase-2

4. Click on it to select ✅

═══════════════════════════════════════════════════════════════════════════

STEP 3: CONFIGURE VERCEL (1 minute)
──────────────────────────────────

1. Project Name:
   → Type: todo-app

2. Root Directory:
   → Click dropdown
   → Select: frontend ✅

3. Click "Deploy" button

4. WAIT... watching for green checkmark (2-3 minutes)

═══════════════════════════════════════════════════════════════════════════

STEP 4: COPY YOUR VERCEL URL (30 seconds)
──────────────────────────────────────────

1. When you see "Congratulations! Your project has been successfully deployed"

2. Copy the URL shown:
   📋 https://[something].vercel.app

3. SAVE THIS URL! (You'll need it) ✅

═══════════════════════════════════════════════════════════════════════════

STEP 5: DEPLOY BACKEND TO RAILWAY (1 minute)
─────────────────────────────────────────────

1. Open in NEW BROWSER TAB:
   👉 https://railway.app

2. Click "Start a New Project"

3. Select "Deploy from GitHub repo"

4. Click "Authorize Railway"

5. Select your GitHub account

6. Choose: hackathin-II-phase-2 ✅

7. WAIT... for green status ✅

═══════════════════════════════════════════════════════════════════════════

STEP 6: COPY YOUR RAILWAY URL (30 seconds)
───────────────────────────────────────────

1. When deployment is done (green status)

2. Click on the project name

3. Go to "Settings" → "Domains"

4. Copy your Railway URL:
   📋 https://[something].railway.app

5. SAVE THIS URL! ✅

═══════════════════════════════════════════════════════════════════════════

STEP 7: CONNECT THEM (1 minute)
───────────────────────────────

1. Go back to Vercel tab

2. Click on your "todo-app" project

3. Click "Settings" (top right)

4. Click "Environment Variables" (left sidebar)

5. Click "Add New"

6. Name: NEXT_PUBLIC_API_URL
   Value: [YOUR RAILWAY URL] (paste it)

7. Click "Save"

8. Go to "Deployments" tab

9. Find latest deployment

10. Click the "..." menu

11. Click "Redeploy"

12. WAIT for green checkmark ✅

═══════════════════════════════════════════════════════════════════════════

STEP 8: TEST YOUR APP (1 minute)
────────────────────────────────

1. Open your Vercel URL in browser:
   https://[your-vercel-url].vercel.app

2. You should see beautiful Todo Dashboard! ✅

3. Try creating a task:
   - Type a title
   - Click "Add Task"
   - Should appear instantly!

4. Refresh the page:
   - Task should still be there! ✅

5. If everything works → YOU'RE LIVE! 🎉

═══════════════════════════════════════════════════════════════════════════

⏱️ TIMER END - YOU'RE DONE!

🎊 YOUR APP IS NOW LIVE!

Share this URL with everyone:
👉 https://[YOUR-VERCEL-URL].vercel.app

═══════════════════════════════════════════════════════════════════════════

IF SOMETHING DOESN'T WORK:

❌ "Can't reach backend"
→ Double-check NEXT_PUBLIC_API_URL in Vercel is correct
→ Make sure Railway deployment shows green status

❌ "Tasks won't save"
→ Open browser DevTools (F12)
→ Check Console for error messages
→ Verify Railway URL is correct

❌ "Vercel/Railway won't deploy"
→ Wait 5 more minutes
→ Check the logs for error messages
→ Restart the deployment
